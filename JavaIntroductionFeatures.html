Java Introduction :

Java is a programming language and a platform. Java is a high level, robust, secured and object-oriented programming language.

Features of Java :

  • Simple
  • Object-Oriented
  • Platform independent
  • Secured
  • Portable
  • Robust
  • Multithreaded
  • Distributed

Applications of Java :

  1. Standalone Application
  2. Web Application
  3. Enterprise Application
  4. Mobile Application

Java Containns Three Tools :

  1. JDK : (Java Development Kit)
    • It contais Development tool such as compiler , debugger and JRE..
  2. JRE : (Java Runtime Environment)
    • It contains Java Packages , Libraries for running the application and JVM..
  3. JVM : (Java Virtual Machine)
    • It helps in executing the java byte code . It have 3 Components..
      1. Classloader :
      2. Memory Area
        • Class Area
        • Heap Area
        • Stack Area
        • PC Register
        • Native Methods
      3. Execution engine
        • Interpreter
        • JIT : (Just In Time)
        • Garbage Collector
